# Настройка кастомного домена giftdraw.today в Vercel

## Проблема
Сайт не загружается по адресу `https://giftdraw.today` потому что домен не добавлен в проект Vercel.

## Решение: Добавление кастомного домена в Vercel

### Шаг 1: Откройте настройки проекта в Vercel

1. Откройте [Vercel Dashboard](https://vercel.com/dashboard)
2. Найдите и откройте проект `crypto-lottery-today`
3. Перейдите в **Settings** → **Domains**

### Шаг 2: Добавьте домен

1. В разделе **Domains** нажмите **Add** или **Add Domain**
2. Введите домен: `giftdraw.today`
3. Нажмите **Add**

### Шаг 3: Настройте DNS записи

Vercel покажет вам DNS записи, которые нужно добавить у вашего регистратора домена.

**Вариант А: Если домен использует Vercel DNS (рекомендуется)**

1. Vercel автоматически настроит DNS записи
2. Просто следуйте инструкциям на экране

**Вариант Б: Если домен использует внешний DNS провайдер**

Вам нужно добавить следующие DNS записи у вашего регистратора домена:

#### Для корневого домена (giftdraw.today):

**A запись:**
- **Type**: A
- **Name**: @ (или оставьте пустым)
- **Value**: IP адрес, который покажет Vercel (обычно `76.76.21.21`)

**ИЛИ CNAME запись:**
- **Type**: CNAME
- **Name**: @ (или оставьте пустым)
- **Value**: `cname.vercel-dns.com`

#### Для поддомена www (www.giftdraw.today):

**CNAME запись:**
- **Type**: CNAME
- **Name**: www
- **Value**: `cname.vercel-dns.com`

### Шаг 4: Ожидание распространения DNS

После добавления DNS записей:
1. Подождите 5-60 минут для распространения DNS изменений
2. Vercel автоматически выдаст SSL сертификат для вашего домена
3. Статус домена в Vercel изменится на "Valid"

### Шаг 5: Проверка

1. Проверьте статус домена в Vercel Dashboard → Settings → Domains
2. Дождитесь статуса "Valid" (зеленая галочка)
3. Откройте `https://giftdraw.today` в браузере
4. Сайт должен загружаться

## Где найти настройки DNS у регистратора домена

### Популярные регистраторы:

1. **Namecheap**: Settings → Advanced DNS
2. **GoDaddy**: DNS Management
3. **Cloudflare**: DNS → Records
4. **Google Domains**: DNS → Custom records
5. **Name.com**: DNS Records

## Важные замечания

1. **HTTPS автоматически**: Vercel автоматически выдаст SSL сертификат через Let's Encrypt
2. **Время распространения DNS**: Обычно 5-60 минут, но может занять до 24 часов
3. **Проверка DNS**: Используйте [whatsmydns.net](https://www.whatsmydns.net) для проверки распространения DNS
4. **Кэш браузера**: Очистите кэш браузера или используйте режим инкогнито

## Если домен все еще не работает

1. **Проверьте DNS записи**: Убедитесь, что записи добавлены правильно
2. **Проверьте статус в Vercel**: Должен быть статус "Valid"
3. **Подождите дольше**: DNS изменения могут занять время
4. **Проверьте регистратора**: Убедитесь, что домен не заблокирован или не на удержании

## Альтернатива: Использование Vercel домена

Если настройка кастомного домена вызывает проблемы, вы можете временно использовать домен Vercel:
- `https://crypto-lottery-today.vercel.app` (или другой домен, который Vercel назначил проекту)

Но для Telegram Login Widget нужен именно кастомный домен `giftdraw.today`.
